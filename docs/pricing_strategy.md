# Pricing Strategy & Packages

## 1. Basic Spark (Landing / Single Page)

**Price**: Starting ₹8,000 - ₹12,000
**Ideal for**: Local businesses, personal brands, event registration.
**Includes**:

- Single high-performance landing page
- Mobile-responsive design
- Contact form integration
- Basic SEO optimization
- Fast delivery (3-5 days)

## 2. Standard Ignite (Full Business Site)

**Price**: Starting ₹22,000 - ₹35,000
**Ideal for**: Corporate entities, medical clinics, service providers.
**Includes**:

- Up to 5 custom-designed pages
- Advanced SEO & PageSpeed optimization
- WhatsApp/Chat integration
- CMS for blog/content updates
- 1 month post-launch support

## 3. Premium Fusion (E-commerce / Web App)

**Price**: Starting ₹45,000+
**Ideal for**: Online stores, custom booking platforms, niche startups.
**Includes**:

- Full E-commerce setup (Product management, Payments)
- Custom user dashboard / Portal
- Advanced animations & premium UX
- API integrations
- 3 months premium support & hosting setup
